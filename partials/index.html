Open source Billing software stack


BillingStack is a set of projects that aim to deliver the needed components to bill services like

  • OpenStack, CloudStack or any other IaaS software stack
  • Phone systems like Asterisk or alike
  • Subscription based systems

Core (Central)

  • Merchants (Tenants)
  • Customers
  • Plans (Rate Plans)

Mediate (Medjatur)

Pull data from any system, mediate it and push to a billingsystem.

Rate (Rater)

Apply pricing on data from a system and turn into a rate.

Bill / Invoice (Biller)

Turn Rate data into Invoice Lines / Items and create Invoices from them.

Payments (Collector)

Collect credits / money for your Invoices using any system
